# FinancApp

Este é um aplicativo de **finanças pessoais** desenvolvido em **React Native** utilizando o framework **Expo**. O projeto faz parte das atividades da disciplina de **Dispositivos Móveis**.

## Funcionalidades

- Cadastro e visualização de receitas e despesas
- Categorização de transações financeiras
- Visualização do saldo atual
- Interface amigável para uso em dispositivos móveis

## Tecnologias Utilizadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [Styled-components](https://styled-components.com/)
- [React Navigation](https://reactnavigation.org/)

## Como rodar o projeto

1. **Clone o repositório**
   ```
   git clone <URL_DO_REPOSITORIO>
   ```
2. **Instale as dependências**
   ```
   npm install
   ```
3. **Execute o aplicativo**
   ```
   npm start
   ```
   Depois, utilize o aplicativo Expo Go no seu dispositivo móvel para escanear o QR Code exibido.

## Requisitos

- Node.js (versão recomendada: >=18)
- Expo CLI (`npm install -g expo-cli`)

## Sobre o Projeto

Este aplicativo foi desenvolvido como parte da disciplina de Dispositivos Móveis, com o objetivo de proporcionar aos alunos experiência prática no desenvolvimento de aplicativos para dispositivos móveis utilizando tecnologias modernas como React Native e Expo.
